Actualmente estoy trabajando en el proyecto personal IsTheWeb. 

Sobre la plataforma OctoberCMS estoy desarrollando e incorporando los módulos y herramientas necesarias para facilitan la creación y gestión de cualquier proyecto tecnológico.
Entre otros destacamos:

  • IsCoorporate: Facilita la gestión de toda la información relevante de una empresa, empleados, proyectos, servicios, galerías, documentos, clientes...
  • IsCommerce: Plataforma de comercio electrónico con productos, categorías, filtros, características ilimitadas, cupones, monedas, sistemas de pago PayPal y Tpv integrados por defecto. Además dispone de un sistema completo de gestión de pedidos, control de inventario...
  • IsMarketing:  La plataforma cuenta con componentes que facilitan la integración de Google Analytics para el seguimiento de contenido y subscriptores de mailings, el protocolo Open Graph para comunicar con redes sociales, Mailing Lists a través de Mailgun API.
